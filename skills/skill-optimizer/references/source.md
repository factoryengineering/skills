# Skill optimizer — sources

This skill applies authoring best practices that come from the open Agent Skills ecosystem. Use these links for the canonical specs, CLI, and documentation.

## Primary source (npm)

**Skills CLI and ecosystem**

- **npm package:** [https://www.npmjs.com/package/skills](https://www.npmjs.com/package/skills)  
  The CLI for the open agent skills ecosystem. Use `npx skills add`, `npx skills list`, `npx skills init`, etc.

- **Repository:** [https://github.com/vercel-labs/skills](https://github.com/vercel-labs/skills)  
  Source for the `skills` npm package.

## Specification and discovery

- **Agent Skills specification:** [https://agentskills.io](https://agentskills.io)  
  Open format for giving agents new capabilities. Defines SKILL.md and how skills are discovered and used.

- **Skills directory:** [https://skills.sh](https://skills.sh)  
  Discover and browse published skills.

## Authoring best practices (canonical)

- **Skill authoring best practices:** [https://platform.claude.com/docs/en/agents-and-tools/agent-skills/best-practices.md](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/best-practices.md)  
  Canonical rules for concise skills, degrees of freedom, progressive disclosure, workflows, content guidelines, anti-patterns, and the checklist. The content in this skill’s [best-practices.md](best-practices.md) is derived from this source.

## Vercel agent skills (example repo)

- **vercel-labs/agent-skills:** [https://github.com/vercel-labs/agent-skills](https://github.com/vercel-labs/agent-skills)  
  Example skills (e.g. react-best-practices, web-design-guidelines) and recommended structure.
